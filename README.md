# 🔐 Secure Password Generator

A simple, fast, and secure desktop application for generating random passwords, built with C++/CLI and Windows Forms (.NET).

## Features
* **Customizable Length**: Choose password length starting from 4 characters.
* **Character Sets**: Toggle uppercase letters, lowercase letters, numbers, and special symbols.
* **One-Click Copy**: Copy the generated password directly to your Windows clipboard with a single click.
* **Validation**: Prompts the user if no character sets are selected.

## Built With
* C++ / CLI
* Windows Forms GUI
* .NET Framework
* Visual Studio 2022

## How to Run
1. Download the latest `.zip` release or clone the repository.
2. If compiling from source, open the `.sln` file in Visual Studio.
3. Build the project using the **Release** configuration.
4. Run the generated `.exe` file!
